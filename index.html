CODAP Data Investigation: Screen Time vs Mental Wellness

In this activity, you will use CODAP to explore how screen time relates to stress, sleep, productivity, and mental wellness.

Dataset: 400 survey participants
Tool: CODAP
Skills: graphing + data moves

Where the data came from

We are using the Mental Wellness & Screen Time Survey (400 Users) dataset from Kaggle. It contains survey responses about daily screen usage and wellness indicators (sleep, stress, productivity, and a mental wellness index).

Source (Kaggle): screentime-vs-mentalwellness-survey-2025

Reminder: This dataset can show patterns (correlations), but it does not prove cause-and-effect.

Column names in our dataset

These are the columns you will see in CODAP:

user_id
age
gender
occupation
work_mode
screen_time_hours
work_screen_hours
leisure_screen_hours
sleep_hours
sleep_quality_1_5
stress_level_0_10
productivity_0_100
exercise_minutes_per_week
social_hours_per_week
mental_wellness_index_0_100

Each row is one participant. Numerical columns are measured as hours, minutes, or scores.

Step 1 — Stress Level vs Screen Time

Goal: Make a scatterplot and describe the relationship.

  1. Create a new Graph in CODAP.
  2. Drag screen_time_hours to the x-axis.
  3. Drag stress_level_0_10 to the y-axis.
  4. Rename the graph: “1. Stress Level vs Screen Time”
  5. Add a note under the plot answering:
  • Do people with higher screen time tend to have higher stress?
  • Are there outliers (points far from most others)?
  • Is there a trend (upward, downward, or no clear pattern)?

Step 2 — Screen Time vs Productivity (with Stress Level)

Goal: Add a third variable using the legend.

  1. Create a new graph.
  2. x-axis: screen_time_hours
  3. y-axis: productivity_0_100
  4. Drag stress_level_0_10 to the Legend.
  5. Rename the graph: “2. Screen Time vs Productivity and Stress Level”
  6. Add a note answering:
  • Is there a relationship between productivity and screen time?
  • Do different stress levels appear in different parts of the graph?

Step 3 — Create “Screen vs Sleep Ratio” + analyze mental wellness

Goal: Make a derived attribute and explore what it means.

3A. Create the derived attribute

  1. In the data table, click the + to add a new attribute.
  2. Name it: Screen_vs_Sleep_Ratio
  3. Choose Formula and enter:
screen_time_hours / sleep_hours
How to interpret Screen vs Sleep Ratio:
  • 1.0 means the person spent the same amount of time on screens as sleeping.
  • 0.5 means the person spent half as much time on screens as sleeping.
  • 2.0 means the person spent twice as much time on screens as sleeping.

3B. Make the graph

  1. Create a new graph.
  2. x-axis: Screen_vs_Sleep_Ratio
  3. y-axis: mental_wellness_index_0_100
  4. Rename the graph: “3. Screen/Sleep Ratio vs Mental Wellness”

3C. Add a distribution + normal curve (for the ratio)

  1. Create a second graph (a distribution plot) for Screen_vs_Sleep_Ratio.
  2. Put Screen_vs_Sleep_Ratio on the axis so you get a dot plot/histogram-like distribution.
  3. In the Measures tab, turn on Normal Curve.
  4. Rename this graph: “3D. Distribution of Screen/Sleep Ratio”

Add a note answering:

  • Did the average survey respondent spend more time on screens or more time sleeping?
  • What was the average Screen_vs_Sleep_Ratio?
  • What relationship do you notice between Screen_vs_Sleep_Ratio and mental_wellness_index_0_100?

Step 4 — Create “Sleep Category” + compare work modes

Goal: Turn sleep hours into categories and compare groups.

4A. Create the derived attribute

  1. Click the + to add a new attribute.
  2. Name it: Sleep_Category
  3. Choose Formula and paste this:
if(sleep_hours < 7, "Deficit",
  if(sleep_hours <= 9, "Ideal", "Extra")
)
Meaning:
  • Deficit = less than 7 hours
  • Ideal = 7–9 hours
  • Extra = more than 9 hours

4B. Make the graph

  1. Create a new graph.
  2. x-axis: work_mode
  3. Legend: Sleep_Category
  4. Rename the graph: “4. Work Mode vs Sleep Deficit”
  5. In the Configuration tab, select:
  • Fuse dots into Bars
  • Percent

Add a note answering:

  • Which work mode was most likely to have Ideal sleep?

Step 5 — Create a new data collection (grouped by work mode) + box plots

Goal: Use data collections to compare mental wellness across work modes.

5A. Create the new data collection

  1. In the data table, drag work_mode to the left side of the table to group by work mode.
  2. CODAP will create a new collection where each row represents a work mode group.

5B. Make box plots comparing mental wellness across work modes

  1. Create a new graph.
  2. x-axis: work_mode
  3. y-axis: mental_wellness_index_0_100
  4. Set the plot type to Box Plot (or use the graph options to display box plots).
  5. Rename the graph: “5. Mental Wellness by Work Mode (Box Plots)”

Add a note answering:

  • Which work mode is most likely to have the highest mental wellness index?
  • Which work mode is most likely to have the lowest mental wellness index?
  • How do you know from the box plots (median, spread, outliers)?

What to turn in

  • Graphs titled exactly as shown in Steps 1–5
  • A short note (1–3 sentences) for each step’s questions
  • Your two derived attributes: Screen_vs_Sleep_Ratio and Sleep_Category
